# DSA Visualizer

An interactive web-based Data Structures and Algorithms Visualizer built using HTML, CSS, and JavaScript. This project helps students and developers understand fundamental DSA concepts through real-time animations and visual representations.

## 🚀 Features

### 🔄 Sorting Visualizer

Visualize and compare popular sorting algorithms with step-by-step animations.

Supported Algorithms:

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort

Features:

* Adjustable array size
* Speed control
* Random array generation
* Pause, resume, and reset functionality
* Real-time complexity display
* Detailed algorithm explanations during execution

---

### 📚 Stack Visualizer

Interactive visualization of the Stack data structure following the LIFO (Last In First Out) principle.

Operations:

* Push
* Pop
* Peek
* Reset
* Random Fill

Features:

* Capacity control
* Overflow and underflow detection
* Live top element tracking
* Animated stack updates

---

### 📥 Queue Visualizer

Interactive visualization of the Queue data structure following the FIFO (First In First Out) principle.

Operations:

* Enqueue
* Dequeue
* Reset
* Random Fill

Features:

* Front and Rear pointer visualization
* Overflow and underflow handling
* Dynamic queue updates
* Capacity management

---

### 🌐 Graph Visualizer

Visualize graph traversal algorithms with animated node exploration.

Supported Algorithms:

* Breadth First Search (BFS)
* Depth First Search (DFS)

Features:

* Select starting node
* Adjustable traversal speed
* Visited order tracking
* Interactive graph layout
* Node state highlighting
* Graph reshuffling

## ⚙️ How to Run

### Method 1: Open Directly

1. Clone the repository

```bash
git clone https://github.com/pragadheeshwar-d/DSA-Visualiser.git
```

2. Navigate to the project folder

```bash
cd DSA-Visualiser
```

3. Open `index.html` in your browser.

---

### Method 2: Using VS Code Live Server

1. Open the project in VS Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.

## 📈 Time Complexities

### Sorting Algorithms

| Algorithm      | Best       | Average    | Worst      |
| -------------- | ---------- | ---------- | ---------- |
| Bubble Sort    | O(n)       | O(n²)      | O(n²)      |
| Selection Sort | O(n²)      | O(n²)      | O(n²)      |
| Insertion Sort | O(n)       | O(n²)      | O(n²)      |
| Merge Sort     | O(n log n) | O(n log n) | O(n log n) |
| Quick Sort     | O(n log n) | O(n log n) | O(n²)      |

### Data Structures

| Operation | Stack | Queue |
| --------- | ----- | ----- |
| Insert    | O(1)  | O(1)  |
| Delete    | O(1)  | O(1)  |
| Peek      | O(1)  | O(1)  |

### Graph Traversal

| Algorithm | Time Complexity | Space Complexity |
| --------- | --------------- | ---------------- |
| BFS       | O(V + E)        | O(V)             |
| DFS       | O(V + E)        | O(V)             |
